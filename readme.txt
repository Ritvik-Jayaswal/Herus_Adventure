Game Title: Heru's Adventure

Game Instructions:
Explore the pyramid using the arrow keys.
Defend youself with a sword by using Z or with a bow/arrow by using X.
Find the stairs on each of the 3 floors to making it to the boss level.
Defeat the boss to earn your pschent (It won't be easy!).

Cheat Codes:
Alt+H: Refill your health
Alt+N: Move to the next floor
Alt+B: Kill the boss, if you are on the boss level
Alt+V: View the entire level (recommended to use Alt+N after this because it is not toggable)
